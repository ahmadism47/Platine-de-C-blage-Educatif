
** Guide d'utilisation du Projet Boîte Groupe **

Bienvenue dans le projet Boîte Groupe ! Ce projet contient les fichiers nécessaires pour assembler les différents composants de la boîte selon leur partie respective.

** Organisation des Fichiers **

Les fichiers sont organisés en fonction des éléments de la boîte groupe, chacun regroupant les pièces qui composent cette partie spécifique. Par exemple :

Le fichier plaque_sortie contient toutes les pièces constituant la plaque de sortie.
Le fichier compartiment_batterie contient les pièces du compartiment de la batterie, et ainsi de suite.
Utilisation du Projet

Pour ouvrir et visualiser le projet dans son intégralité, veuillez suivre les étapes suivantes :

Téléchargez tous les fichiers du projet.
Ouvrez le produit sous le nom "Platine_Assemblage" à l'aide du logiciel CATIA.
En ouvrant ce fichier, le produit complet de la boîte groupe apparaîtra, avec tous ses composants assemblés.
N'hésitez pas à explorer les différents fichiers pour découvrir les pièces individuelles et leur organisation dans le projet.

Remarque : Assurez-vous d'avoir installé le logiciel CATIA sur votre ordinateur pour ouvrir et visualiser les fichiers du projet.
